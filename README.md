<b>Installation:</B>

1. Go to the Kodi <b>File Manager</b>
2. Click on <b>Add source</b>
3. The path for the source is https://konrepo.github.io/KhmerKodi then click OK
4. Go back to </b>Add-ons</b>
5. In Add-ons, select <b>Install add-on from zip</b>
6. When asked for the location, select [SourceName], and install repository.KhmerKodi.zip
7. Go back to Add-ons install, but this time, select <b>Install from repository</b>
8. Select the <b>KhmerKodi</b>
9. Go into the <b>Video add-ons</b> section in the repo, and select KhmerKodi
